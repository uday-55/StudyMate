# **App Name**: StudyMate

## Core Features:

- PDF Q&A: Enables users to upload PDFs and ask questions in natural language, with context-aware answers powered by an LLM.
- Summarization Tool: Generates quick or detailed summaries of entire PDFs or specific sections using a generative AI tool, allowing users to efficiently grasp key concepts.
- AI Flashcard Generator: Automatically creates question/answer pairs from the PDF content, and allows exporting them.
- Chat Interface: Features a chat-like interface with file upload, voice input/output, dark/light theme toggle, and typing animations.
- Concept Map Visualization: Generates a visual knowledge graph displaying relationships between topics within the PDF content.
- Quiz Generation Tool: Auto-generates quizzes from study material, categorizing questions by difficulty level with the help of a generative AI tool.

## Style Guidelines:

- Primary color: Deep blue (#3168BD) to evoke a sense of trust and intellectual depth, given the app's focus on study materials.
- Background color: Very light blue (#E6EEF9), a desaturated shade of the primary color, providing a calming backdrop suitable for reading.
- Accent color: Soft green (#8FCE00), an analogous color to the primary blue, adding contrast and highlighting key interactive elements such as buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Code font: 'Source Code Pro' for displaying any code snippets or technical examples.
- Use clear, consistent icons from shadcn/ui to represent different functions and file types.
- Employ subtle animations powered by Framer Motion for transitions and feedback, creating a smooth and engaging user experience.
